Panduan

   tutorial penggunaan!

1.penambahan prompt

kamu bisa mengetik prompt tambahan seperti :
(venn dan Elaina adalah suami istri dan kini Elaina sedang tidur di dada venn saat malam hari)

note: perlu diperhatikan. tidak semua character menerima prompt tambahan

2.simbol teks

simbol teks hanya menggunakan "({text})" untuk narasi contohnya:
(venn dan Elaina adalah suami istri dan kini Elaina sedang tidur di dada venn saat malam hari)

3.gambar

2.1 untuk membaca gambar kamu harus mengirim gambar > lalu ketik perintah seperti :"ini apa?" lalu kirim

2.2 untuk membuat gambar dengan prompt saat ini tidak ada karena fitur berbayar

4.hapus percakapan

hapus percakapan bukan menghapus 1 chat melainkan hapus sesi chat dan tidak bisa diulang
